# Meta SDK use

Meta SDK libraries used for Spatial Lingo:

- [Audio](https://developers.meta.com/horizon/documentation/unity/meta-xr-audio-sdk-unity/)
- [Environment Raycast](https://developers.meta.com/horizon/documentation/unity/unity-mr-utility-kit-environment-raycast)
  - See [CameraTaxonTracker.cs](../Packages/com.meta.utilities.taxontracking/Runtime/Scripts/CameraTaxonTracker.cs) for more.
- [Hand Tracking](https://developers.meta.com/horizon/documentation/unity/unity-handtracking-overview/)
- [Interaction](https://developers.meta.com/horizon/documentation/unity/unity-isdk-interaction-sdk-overview/)
  - See [HandTrigger.cs](../Assets/SpatialLingo/Scripts/Interactions/HandTrigger.cs) for more.
- [Llama](https://www.llama.com/)
- [Mixed Reality Utility Kit (MRUK)](https://developers.meta.com/horizon/documentation/unity/unity-mr-utility-kit-overview/)
- [Passthrough](https://developers.meta.com/horizon/documentation/unity/unity-passthrough)
- [Voice](https://developers.meta.com/horizon/documentation/unity/voice-sdk-overview)
  - See [VoiceSynthesis.md](../Packages/com.meta.utilities.speechandtext/Documentation~/VoiceSynthesis.md) and [VoiceTranscription.md](../Packages/com.meta.utilities.speechandtext/Documentation~/VoiceTranscription.md) for more.
- [XR Camera Rig](https://developers.meta.com/horizon/documentation/unity/unity-ovrcamerarig/)
