# Meta Speech and Text

## Intro

This package encapsulates the Speech Dictation and Text to Speech (TTS) capabilities of wit.ai.

## Installation

You can integrate this package into your own project by using the Package Manager to [add the following Git URL](https://docs.unity3d.com/Manual/upm-ui-giturl.html):

```txt
https://github.com/oculus-samples/Unity-SpatialLingo.git?path=Packages/com.meta.utilities.speechandtext
```
